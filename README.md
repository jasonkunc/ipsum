# Vegan Ipsum Generator

A Lorem Ipsum generator that creates placeholder text using plant-based words instead of Latin.

## Features

- **200+ Vegan Words** - Vegetables, fruits, nuts, legumes, grains, herbs, fungi, and plant-based terms
- **Smart Generation** - Random paragraph lengths (10-100 words) for natural variety
- **Customizable** - Choose 1-20 paragraphs (default: 7)
- **Random Backgrounds** - Each page refresh has a unique color with automatic text contrast
- **Neobrutalist Design** - Bold borders, heavy shadows, and high contrast
- **Responsive** - Works on all screen sizes
- **Keyboard Shortcuts** - Ctrl/Cmd + Enter to generate, Ctrl/Cmd + C to copy

## Usage

1. Enter number of paragraphs (1-20)
2. Click "GENERATE IPSUM" or press Ctrl/Cmd + Enter
3. Copy text with "COPY TEXT" button or Ctrl/Cmd + C

## Tech

- Vanilla JavaScript (no dependencies)
- Modular word database
- Responsive CSS with neobrutalist styling
- GitHub Pages ready
